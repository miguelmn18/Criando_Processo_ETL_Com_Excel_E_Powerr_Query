# Criando_Processo_ETL_Com_Excel_E_Powerr_Query

Objetivo das aulas fornecidas pelo Bootecamp Da DIO foi melhorar as informações fornecidas por uma empresa fictícia. 
Pegar uma planilha e organizar os dados.
Através de funcionalidades que o Excel agora fornece, não é mais preciso criar programas para organizar as planilhas.

Organização: Apagar as linhas em branco, realizar cálculos, mudar a ordem de uma coluna, alterar palavras de maiscúlo para minúsculo, traduzir do japonês para o português.
